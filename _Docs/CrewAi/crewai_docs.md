# CrewAI Documentation

## Introduction
CrewAI is a framework for building collaborative AI agents.

### Core Concepts
- **Agents**: Autonomous units with roles and goals.
- **Tasks**: Specific assignments for agents.
- **Crews**: Groups of agents working together.
- **Flows**: Orchestration of agents and tasks.

### Installation
```bash
uv pip install crewai
```
or
```bash
pip install crewai
```

### Quickstart
See [CrewAI Quickstart](https://docs.crewai.com/en/quickstart) for more details.

## Key Features
- **Role-Based Agents**: Customize agents with specific roles, goals, and backstories.
- **Autonomous Inter-Agent Delegation**: Agents can delegate tasks and inquire amongst themselves.
- **Flexible Task Management**: Define tasks with customizable tools.
- **Process Driven**: Supports sequential, hierarchical, and consensual processes.

## Enterprise Features
- **Deploy Automations**: Manage environments and monitor runs.
- **Triggers**: Connect with Gmail, Slack, Salesforce, etc.

## Integrations
- **LangChain**: Seamless integration with LangChain tools.
- **Observability**: Integration with Arize Phoenix, Langfuse, MLflow.
